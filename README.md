# Backend API Server
REST API Backend built using Node.js, Express.js, MongoDB, Mongoose, Postman, and Render deployment.

---

## Features

- Create User API
- Login User API
- Add Student API
- Get All Users API
- MongoDB Database Integration
- REST API Architecture
- Postman API Testing
- Render Deployment

---

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Postman
- Render
- GitHub

---

## API Endpoints

### Home Route

GET /

### Create User

POST /create-user

### Login User

POST /login

### Add Student

POST /add-student

### Get All Users

GET /users

---

## Installation

### Install Dependencies

```bash
npm install
```

### Run Server

```bash
node server.js
```

---

## Database

MongoDB Atlas is used as the cloud database for storing user and student data.
